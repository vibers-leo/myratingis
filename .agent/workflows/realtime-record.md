# 실시간 세션 기록 모드

이 워크플로우가 활성화되면 대화 종료 시 자동으로 세션 로그를 저장합니다.

## 기록 규칙

1. **파일 위치**: `session-logs/YYYY-MM-DD-한글제목.md`
2. **파일 형식**: 한글 제목, 한글 내용
3. **필수 포함 내용**:
   - 날짜 및 작업 시간
   - 주요 작업 요약
   - 수정된 파일 목록
   - 커밋 이력
   - 기술적 인사이트
   - 다음 단계 권장 사항

## 자동 기록 트리거

대화 종료 시 또는 사용자가 "세션 저장" 요청 시 실행됩니다.
