### 2단계: 인증 시스템 개선 (2025-12-23 ~ 2025-12-24)

#### Q5: 새로고침 시 로그인 풀림 문제

**문제**: 사용자가 "로그인이 되었다가 갑자기 새로고침되서 관리자 로그인이 풀렸어"

**원인 분석**:

1. `AuthContext`의 초기화 로직에서 `getSession`과 `onAuthStateChange`가 경쟁 상태(Race Condition) 발생
2. `TOKEN_REFRESHED` 이벤트 처리 시 상태 초기화 문제
3. 타임아웃 로직 부재로 무한 로딩 가능성

**해결책**:

```typescript
// AuthContext.tsx 개선
const initializeAuth = async () => {
  try {
    // 5초 타임아웃 안전장치 추가
    const timeoutPromise = new Promise<{
      data: { session: Session | null };
      error: any;
    }>((_, reject) =>
      setTimeout(() => reject(new Error("Auth Initialization Timeout")), 5000)
    );

    const {
      data: { session: currentSession },
      error,
    } = await Promise.race([supabase.auth.getSession(), timeoutPromise]);

    // ... 세션 처리
  } catch (error) {
    console.error("[Auth] 초기화 예외 발생:", error);
  } finally {
    if (isMounted) setLoading(false); // 무조건 로딩 해제
  }
};
```

**주요 개선 사항**:

1. `isAdmin` 상태를 별도 `useState`로 분리하여 상태 불일치 방지
2. `initializeAuth`에 `Promise.race`로 5초 타임아웃 적용
3. `finally` 블록에서 무조건 `setLoading(false)` 호출
4. `onAuthStateChange` 이벤트 핸들러 정리

**커밋**: `103dc19` - "refactor: improve session initialization and admin check in AuthContext for stability"

#### Q6: 관리자 페이지 무한 로딩 문제

**문제**: "관리자 페이지가 안들어가지네유~~" - 무한 로딩 스피너

**원인**:
`useAdmin` 훅이 `AuthContext`와 독립적으로 동작하며 자체 타임아웃 및 DB 조회 수행 → 상태 불일치 및 중복 네트워크 요청

**해결**:

```typescript
// Before: 독립적인 로직
export function useAdmin(): AdminState {
  const [state, setState] = useState<AdminState>({ ... });

  useEffect(() => {
    const checkAdmin = async () => {
      // 독자적인 DB 조회 및 타임아웃 로직
    };
    checkAdmin();
  }, []);

  return state;
}

// After: AuthContext 기반
export function useAdmin(): AdminState {
  const { isAdmin, loading, user, userProfile } = useAuth();

  return {
    isAdmin,
    isLoading: loading,
    userId: user?.id || null,
    userRole: userProfile?.role || (isAdmin ? 'admin' : 'user'),
  };
}
```

```typescript
import { CustomImageExtension } from "./CustomImageExtension";

// 이미지 교체 이벤트 리스너 추가
useEffect(() => {
  if (!editor) return;

  const handleReplaceImage = async (event: Event) => {
    const customEvent = event as CustomEvent;
    const { file, pos } = customEvent.detail;

    if (file && typeof pos === "number") {
      const url = await uploadImage(file, "projects");
      editor.commands.updateAttributes("customImage", { src: url });
    }
  };

  const editorElement = editor.view.dom;
  editorElement.addEventListener(
    "replaceImage",
    handleReplaceImage as EventListener
  );

  return () => {
    editorElement.removeEventListener(
      "replaceImage",
      handleReplaceImage as EventListener
    );
  };
}, [editor]);
```
